# Assets

에셋 정리용 폴더. 랜딩페이지 리디자인에 사용.

## 구조

```
photos/
  hero/           히어로 키비주얼 (후보 2-3장)
  empathy/        공감 섹션 (일정/피팅/검진 분위기)
  offer/          혜택 섹션 (촬영·드레스 실컷)
  testimonials/   후기 섹션 (인물 or 현장)
  planner/        플래너 프로필
brand/
  logo/           로고 (svg 최우선, png도 ok)
  colors.md       브랜드 컬러 hex (있으면)
  guideline.pdf   브랜드 가이드 문서 (있으면)
copy/
  testimonials.md    실제 후기 텍스트 (있으면)
  reference-urls.md  기존 다이렉트결혼준비 URL
```

## 파일명 규칙
- 공백 대신 `-` 또는 `_` 사용
- 한글 파일명 OK
- 가능하면 설명 포함 (예: `hero-01-couple-outdoor.jpg`, `testimonial-kim-ji-eun.jpg`)

## 사진 권장 사양
- **가로**: 최소 1920px, 세로 최소 1280px 선호
- **포맷**: jpg (사진) / png (투명 로고) / svg (로고 우선)
- 보정된 원본 또는 고화질이면 좋음. 제가 필요에 맞게 크롭·리사이즈 함
