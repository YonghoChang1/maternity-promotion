# brand/

브랜드 자산.

## logo/
- **svg 최우선** (벡터, 색 조정 자유)
- png는 투명 배경
- 가로형/세로형/심볼형 여러 버전 있으면 다 넣어주세요
- 밝은 배경용 / 어두운 배경용 따로 있으면 구분 (예: `logo-dark-bg.svg`)

